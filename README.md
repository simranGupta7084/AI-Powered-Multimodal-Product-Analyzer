🚀 AI-Powered Multimodal Product Analyzer & Comparator

This project is a multimodal AI web application that takes a product image as input and generates intelligent insights such as product description, estimated price, and sentiment-based reviews. It also compares outputs from multiple AI models like OpenAI GPT-4o and Google Gemini, providing a benchmarking dashboard for performance evaluation.

🔥 Features
🖼️ Image-based product analysis
🤖 AI-generated product descriptions
💰 Price estimation using ML models
💬 Sentiment-aware review generation
⚖️ Multi-model comparison (GPT-4o vs Gemini)
📊 Evaluation dashboard (response time, quality metrics)
🌐 Interactive UI using Streamlit
🛠️ Tech Stack
Frontend: Streamlit
Backend: Python
AI Models: OpenAI GPT-4o, Google Gemini
Machine Learning: Scikit-learn
Database: SQLite
